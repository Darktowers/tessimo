La tecnología Intellibrew™ de TASSIMO te asegura el sorbo perfecto en todos los sentidos.

Tanto si se trata de un espresso, un té de hierbas revitalizante o un chocolate caliente ultracremoso, TASSIMO prepara cada bebida de manera individual para obtener los resultados más deliciosos con tan solo pulsar un botón.

La máquina escanea el código de barras de cada T DISC para obtener la información necesaria: la cantidad correcta de agua, el tiempo exacto de elaboración y la temperatura ideal para preparar tu bebida preferida. TASSIMO hace todo el trabajo, dejándote tiempo para disfrutar de momentos especiales a cualquier hora del día.
